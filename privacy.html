Privacy Policy

1. General Information

This Privacy Policy describes how your personal information is handled in this PhoneBook web application. This project is created for educational purposes within the "Web Design" course.

2. Data Collection

We collect information that you voluntarily provide through the registration and profile forms, specifically:

  • Full Name
  • Email Address
  • Gender
  • Date of Birth
3. Data Usage & Storage

This is a static web application. No data is transmitted to any remote server or stored in a permanent database. Any data entered into the forms is processed locally within your browser and will be cleared upon refreshing the page.

4. Purpose of the Project

The sole purpose of this site is to demonstrate front-end development skills using HTML5, CSS3, and the Bootstrap framework. It is not used for any commercial or marketing activities.
